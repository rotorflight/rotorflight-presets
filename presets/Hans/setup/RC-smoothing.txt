#$ DESCRIPTION: This preset combines RC command smoothing with rate response tuning. It is designed for fast, low-latency links such as ELRS, FrSky Twin, Tandem, and Archer systems.
#$ DESCRIPTION: Rather than altering PID gains, this preset reshapes the RC command signal before it is processed by the PID controller.
#$ DESCRIPTION: The goal is to refine how stick inputs translate into rotor response, making the helicopter feel more predictable and polished without changing the core tune.

#$ DESCRIPTION: Two options are included:
#$ DESCRIPTION: • Crisp – A sharper, more immediate connection to the model with minimal input softening.
#$ Description: • Polished – Smoother transitions and a more composed, refined feel in fast directional changes and sustained maneuvers.

#$ DESCRIPTION: This preset enhances feel on mechanically sound, properly tuned helicopters. It does not replace correct PID setup.


#$ PRIORITY: 0
#$ FORCE_OPTIONS_REVIEW: TRUE

    set yaw_response = 4
    set rc_smoothness = 60
